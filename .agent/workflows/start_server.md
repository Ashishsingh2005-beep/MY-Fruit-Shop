description: Start the backend server for the Fruit Shop application
---

1. Open the terminal.
2. Navigate to the project directory: `c:\Users\Admin\OneDrive\Desktop\Fruit shop`
3. Run the server command:
   ```bash
   python backend/server.py
   ```
4. Verify the server is running by checking for output like:
   ```
   * Serving Flask app 'server'
   * Debug mode: on
   * Running on http://127.0.0.1:5000
   ```
